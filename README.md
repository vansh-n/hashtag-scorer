# Hashtag Scorer

A free, open-source Instagram hashtag scoring tool for any niche.

## What It Is
A web-based tool that analyzes and scores social media hashtags based on three weighted factors: search volume, competition level, and niche relevance. Results are output in three clear tiers so you know exactly which hashtags to prioritize.

## How It Works
1. Enter your niche (e.g. "swim coach for teens", "vegan food blogger", "real estate agent")
2. Select your account size
3. Select your primary goal — Reach, Engagement, or Followers
4. Get a scored, tiered list of hashtags ranked 0–100

## The Three Tiers
- 🟢 **High Leverage** — Low competition, high relevance. Best chance of hitting top posts.
- 🟡 **Broad Reach** — High volume but competitive. Good for discovery.
- 🔵 **Niche Anchors** — Small, hyper-targeted tags. Easy to dominate.

## Scoring Algorithm
Each hashtag is scored using a weighted composite of:
- **Search Volume** — how many posts use this tag
- **Competition Level** — how hard it is to rank in top posts
- **Niche Relevance** — how closely it matches your input
- **Account Size Modifier** — small accounts get boosted toward low-competition tags, growing accounts toward high-volume tags
- **Unique Micro-Score** — ensures every hashtag receives a distinct score
